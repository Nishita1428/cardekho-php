A CarDekho-inspired web application built using PHP and MySQL.
The project allows users to submit car enquiries and provides a secure admin panel to view all enquiries.

---

TECH STACK-->
- HTML5
- CSS3
- PHP
- MySQL
- phpMyAdmin
- XAMPP (Localhost)



FEATURES

USER-SIDE
- Responsive homepage inspired by CarDekho
- Car Enquiry Form
- Multiple car type selection (Hatchback, Sedan, SUV)
- Enquiry data stored in MySQL database

ADMIN-SIDE
- Secure admin login using PHP sessions
- Password hashing with `password_hash()` and verification using `password_verify()`
- Admin dashboard to view all enquiries
- Logout functionality (session destroy)
- Admin pages protected from unauthorized access



DATABASE
- CAR-ENQUIRY: Stores user enquiries
- ADMIN: Stores admin credentials (hashed passwords)
